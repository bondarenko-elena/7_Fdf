# Fdf
This project is about creating a simplified 3D graphic representation of arelief landscape with minilibX library.

Hooks for MacOS: </br>
relief +/-               ---> Numpad 6 / Numpad 9 </br>
change color             ---> C </br>
zoom +/-                 ---> Numpad + / Numpad - </br>
move up/down/left/right  ---> arrows </br>
